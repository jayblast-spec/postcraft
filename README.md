## ✨ Features

- 🎭 **4 tone modes** — Professional, Casual, Storytelling, Controversial — pick what fits your brand
- 🪝 **Hook-first writing** — every post opens with a proven scroll-stopping structure
- 🔀 **Dual-platform output** — optimized separately for LinkedIn (long-form) and X (punchy)
- 💡 **Topic → post in 10 seconds** — paste an idea, get a polished post immediately
- 🔄 **Regenerate variants** — not vibing with the first draft? Try again with one click
- 📋 **Copy to clipboard** — paste straight into the platform composer
- ⚡ **No signup, no paywall** — open it, use it, close it

## 🚀 Quick Start

```bash
git clone https://github.com/jayblast-spec/postcraft.git
cd postcraft
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
